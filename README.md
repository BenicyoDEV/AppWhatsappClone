# 📱 WhatsApp Clone (React Native)

Projeto desenvolvido durante as aulas da **ETEC de Poá**, com foco nos estudos de **React Native** e desenvolvimento mobile.
A aplicação simula a interface do WhatsApp, reproduzindo suas principais abas e elementos visuais.

**📅 Data de produção:** Junho de 2026

---

## 📱 Sobre o Projeto

O aplicativo recria a interface principal do WhatsApp, com foco em UI/UX e organização de telas.  
Ele é dividido em quatro áreas principais:

- Conversas (Chat principal)
- Status (Atualizações)
- Comunidades
- Ligações

Cada tela foi construída manualmente com componentes do React Native, simulando o comportamento visual do app real.

---

## 🧩 Tecnologias

- React Native – Desenvolvimento mobile  
- Expo – Ambiente de execução  
- Expo Router – Navegação por abas e páginas  
- StyleSheet – Estilização das interfaces  
- React Navigation (Tabs) – Sistema de navegação inferior  

---

## 🎯 Objetivo do Projeto

O projeto foi desenvolvido com foco em:

- Aprender React Native na prática
- Entender navegação com Expo Router
- Reproduzir interfaces reais de aplicativos grandes
- Evoluir habilidades de layout com Flexbox
